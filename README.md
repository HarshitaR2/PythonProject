An in-depth data analysis project exploring U.S. public library operations, usage patterns, and financial health. This project uncovers key insights about how population, funding, and digital engagement influence library performance.
Conducted comprehensive EDA on public library datasets across the U.S.
Examined patterns in library usage, service trends, financial stability, and digital engagement.
Analyzed relationships between population served, expenditures, and service efficiency using Python, Pandas, and NumPy.
Created impactful visualizations including scatter plots, heatmaps, correlation matrices, and trend graphs with Matplotlib and Seaborn.
Delivered actionable insights to understand operational strengths and improvement areas within public libraries.
 Tech Stack-Python, Pandas, NumPy,Matplotlib, Seaborn,Data analysis & visualization
